思路：不妨记y=y1,dy1dt=y2,dy2dt=d2ydt2,那么有{dy1dt=y2dy2dt=7∗(1−y12)∗y2−y1

那么这样我们就把一个二阶微分方程拆成了两个一阶微分方程，所以需要同时进行两个龙格库塔迭代来进行计算

程序：

xxxxxxxxxx
% 表示微分方程组
function dy=Fun(x,y)
dy=zeros(size(y));
dy(1)=y(2); %dy(1)表示以y的一阶导为f(x,y)
dy(2)=7*(1-y(1).^2)*y(2)-y(1);  %dy(2)表示以y的二阶导为f(x,y)
xxxxxxxxxx
% 龙格库塔法
function y=RK(t,h,y)


len = length(t);


for i=2:len         %循环：直到求完最后一个t取值


K1 = Fun(t(i-1),y(i-1,:));          % K1




K2 = Fun(t(i-1)+1/2*h , y(i-1,:)+1/2*h.*K1);        % K2




K3 = Fun(t(i-1)+1/2*h , y(i-1,:)+1/2*h.*K2);        % K3




K4 = Fun(t(i-1)+h , y(i-1,:)+h.*K3);        % K4




y(i,:) = y(i-1,:)+h/6.*(K1 + 2*K2 + 2*K3 + K4);         
                                                    
end


xxxxxxxxxx
%求解过程
t=[0 : 0.01 :20]; % t——行向量
h=0.01; % 步长
y=zeros(length(t) , 2); 
y(1,:)=[1.0 , 0.0]; % y的初值1.0 ，y一阶导的初值 0.0
% 函数调用
Y=RK(t,h,y); % 调用4阶R-K法，输入参数t，h，y
%画图

利用阻滞增长模型（Logistic模型）进行人口预测，设增长率为r ,在人口较少时可以把r 看作常数，当人口增加到一定数目时r 视为一个随人口增加而减少的变量，即增长率r 表示为人口x(t)的函数r(x)，且r(x)为x减函数 。

设人口上限为xm，r(x)=r−sx(r,s>0),r 固有增长率(x很小时),又r(xm)=0⇒s=rxm⇒r(x)=r(1−xxm)

建立微分方程：

{dxdt=r(x)x=rx(1−xxm)x(t0)=x0⇒x(t)=xm1+(xmx0−1)e−rt

接下来利用非线性最小二乘法估计r,xm， 

xxxxxxxxxx
% 已知数据
tdata = 1810:10:2010;
xdata = [3.9,5.3,7.2,9.6,12.9,17.1,23.2,31.4,38.6,50.2,62.9,76.0,92.0,106.5,123.2,131.7,150.7,179.3,204.0,226.5,251.4];


% 定义要拟合的函数
fun = @(x, t) x(1) ./ (1 + (x(1) / xdata(1) - 1) * exp(-x(2) * (t - 1810))); % xm=x(1) r=x(2)


% 初始参数猜测

七：

利用最小二乘法计算a、b、c、d、e、值：

xxxxxxxxxx
% 输入数据
xdata = [5.764 6.286 6.759 7.168 7.408];
ydata = [0.648 1.202 1.823 2.526 3.360];
syms a b c d e;


% 计算残差
T = a * xdata.^2 + b * ydata.^2 + c * xdata .* ydata + d * xdata + e * ydata + 1 - ydata;
delta = sum(T.^2);


% 对 delta 分别对 a, b, c, d, e 求偏导数
partial_a = diff(delta, a);
partial_b = diff(delta, b);
partial_c = diff(delta, c);
partial_d = diff(delta, d);
partial_e = diff(delta, e);


% 联立求解
solutions = solve([partial_a == 0, partial_b == 0, partial_c == 0, partial_d == 0, partial_e == 0], [a, b, c, d, e]);


% 转换解为小数形式
a = double(solutions.a);
b = double(solutions.b);
c = double(solutions.c);
d = double(solutions.d);
e = double(solutions.e);


% 输出结果保留四位小数
fprintf('a: %.4f\n', a);
fprintf('b: %.4f\n', b);
fprintf('c: %.4f\n', c);
fprintf('d: %.4f\n', d);
fprintf('e: %.4f\n', e);


% 绘制隐式方程的图形
fimplicit(@(x, y) a * x.^2 + b * y.^2 + c * x .* y + d * x + e * y + 1, [-100,100]);
title('隐式方程图形');
xlabel('X轴');
ylabel('Y轴');
